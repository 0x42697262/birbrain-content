= Headscale

An open-source control server for xref:tailscale.adoc[].

== Users

=== Registering new users

.https://headscale.net/stable/usage/getting-started/#create-a-headscale-user
----
headscale users create <USER>
----

=== Listing existing users

.https://headscale.net/stable/usage/getting-started/#list-existing-headscale-users
----
headscale users list
----

== Nodes

=== Listing Nodes

----
headscale nodes list
----

=== Routes

----
headscale nodes list-routes
----

== Using a preauthkey

.https://headscale.net/stable/usage/getting-started/#list-existing-headscale-users
----
headscale preauthkeys create --user <USER_ID>
----

Use the ID provided in the list of users.

=== Expiration

----
headscale preauthkeys create --user <USER_ID> --expiration 1y
----
