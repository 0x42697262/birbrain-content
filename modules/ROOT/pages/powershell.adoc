= Powershell

== History

=== Print
----
(Get-PSReadlineOption).HistorySavePath
----

=== Delete

----
Remove-Item (Get-PSReadlineOption).HistorySavePath
----

== Remote Desktop Session

=== Shadow a User

----
mstsc /shadow:# /control
----

NOTE: Use `quser` to get the user id.

NOTE: If logged in as admin, there is no need for a password.
