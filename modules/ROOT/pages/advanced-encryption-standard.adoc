= Advanced Encryption Standard

The Advanced Encryption Standard (AES) is a symmetric block cipher standardized by
NIST in FIPS 197 (2001), based on the Rijndael cipher. It encrypts data in 128-bit
blocks using a key of 128, 192, or 256 bits, and is the most widely used symmetric
cipher today (TLS, disk encryption, Wi-Fi, etc.).
